Search, read, and export papers from your Zotero library

Get Started View on GitHub

Everything you need

A complete toolkit for working with your research library from the command line

🔍

Full-text Search

Search across titles, authors, years, and abstracts. Filter by tags, date ranges, and item types.

🧠

Semantic Search

Natural language queries powered by ChromaDB embeddings. Find papers by meaning, not just keywords.

📖

Read as Markdown

Convert PDFs to clean markdown. Paginate, search within, extract figures and tables.

📋

BibTeX Export

Export citations with configurable field filtering. Ready for LaTeX, Typst, or any reference manager.

👥

Group Libraries

Access shared Zotero group collections. Search across all libraries at once with --all-libraries.

🤖

AI Agent Ready

Built-in Claude Code skill for paper research workflows. Let agents search and read your library.

Install in seconds

Requires Zotero desktop running with the local API enabled

$ uv tool install riszotto
$ riszotto --help

# With semantic search
$ uv tool install "riszotto[semantic]"
$ pip install riszotto
$ riszotto --help

# With semantic search
$ pip install "riszotto[semantic]"
# Install as a Claude Code skill
$ npx skills add https://github.com/pythonfz/riszotto

# Then ask Claude to search your library
